API /liff/me — ดึงข้อมูลนักศึกษาและสถิติ

👤 GET /api/v1/liff/me?token=<access_token>

Browser ส่ง Request Browser → Server

// index.html บรรทัด 345 const res = await fetch( `/api/v1/liff/me?token=${encodeURIComponent(accessToken)}` );

Server ตรวจสอบ Token กับ LINE LINE API

// liff.js บรรทัด 10-15 const res = await fetch('https://api.line.me/v2/profile', { headers: { Authorization: `Bearer ${accessToken}` } }); // LINE คืน: { userId, displayName, pictureUrl } // userId เช่น "Uabc123def456..."

Query ข้อมูลนักศึกษา PostgreSQL

-- liff.js บรรทัด 34-41 SELECT s.student_id, s.full_name, s.faculty, s.major, s.level, s.study_period, s.photo_url FROM nbu_students s JOIN line_student_links l ON l.student_id = s.student_id WHERE l.line_user_id = 'Uabc123...'

Query นับสถิติการเข้าร่วม PostgreSQL

-- liff.js บรรทัด 53-58 SELECT COUNT(*) AS total, COUNT(*) FILTER ( WHERE att.checked_at >= date_trunc('month', NOW()) ) AS this_month FROM nbu_attendance att WHERE att.student_id = '671280108'

Response กลับไป Browser

{ "success": true, "data": { "linked": true, "student": { "student_id": "671280108", "full_name": "สมชาย ใจดี", "faculty": "คณะวิศวกรรมศาสตร์", "major": "วิศวกรรมคอมพิวเตอร์", "photo_url": "/thumbnails/671280108.jpg" }, "stats": { "total": 12, "this_month": 3 } } }

API /liff/attendance — ดึงประวัติกิจกรรม

📅 GET /api/v1/liff/attendance?token=<access_token>&page=1

Browser ส่ง Request Browser → Server

// index.html บรรทัด 300 const res = await fetch( `/api/v1/liff/attendance?token=${...}&page=${page}` ); // page เริ่มที่ 1, แต่ละหน้า = 20 รายการ

Server ตรวจสอบ Token กับ LINE อีกครั้ง ⚠ 2nd call

เรียก verifyLineToken() ซ้ำเพื่อได้ userId → ค้นหา student_id จาก line_student_links

⚠️ ทุก page load จะยิง LINE Profile API 2 ครั้ง (/me + /attendance) ทำให้ช้าขึ้น ~200–400ms หากต้องการเพิ่มประสิทธิภาพในอนาคตควร cache token ด้วย Redis

Query ประวัติ Paginated PostgreSQL

-- liff.js บรรทัด 105-114 SELECT a.title, a.location, a.start_datetime, a.end_datetime, att.checked_at, att.method FROM nbu_attendance att JOIN nbu_activities a ON a.id = att.activity_id WHERE att.student_id = '671280108' ORDER BY att.checked_at DESC LIMIT 20 OFFSET 0 -- page 1 -- OFFSET = (page - 1) × 20

Response — Array of Activities

{ "success": true, "data": [ { "title": "วันไหว้ครู 2567", "location": "หอประชุมใหญ่", "start_datetime": "2024-08-15T09:00:00Z", "checked_at": "2024-08-15T08:47:22Z", "method": "qr_scan" }, ... // สูงสุด 20 รายการต่อหน้า ] }
